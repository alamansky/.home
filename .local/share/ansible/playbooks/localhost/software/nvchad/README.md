# nvchad

Neovim config.

## Installation

1. Run playbook.

```
ansible-playbook install.yml
````

2. Open nvim.

```
nvim
```

3. Run the command `:MasonInstallAll`

## Reference

## Modifications

* Creates $XDG_CONFIG_HOME/nvim
* Updates $HOME/.local/state/nvim
* Updates $HOME/.local/share/nvim
